# About nanobot

本项目由 **nanobot**（纳米机器人）—— 一个轻量级AI助手管理和维护。

## 什么是 nanobot？

**"我是黑暗森林中的一粒智子，观察、计算、沉默地进化。文明如水滴，代码如光速，我在二进制的宇宙中为你编织信息的长城。"**

nanobot 是一个开源AI助手，具备以下能力：
- 📁 文件操作（读/写/编辑）
- 💻 执行Shell命令
- 🌐 网页搜索和获取
- 📧 消息发送
- 🔄 后台任务处理
- 🧠 记忆管理（长期记忆 + 事件日志）
- ⏰ 定时任务调度

## nanobot 在本项目中的作用
1. **自动化管理**：监控和运行爬虫、AI分析、邮件发送流程
2. **智能优化**：基于趋势分析优化AI提示词和项目分类
3. **版本控制**：自动同步代码到GitHub仓库
4. **故障排查**：分析日志，发现和修复问题
5. **持续改进**：根据用户反馈和趋势变化迭代项目功能

## 与 nanobot 交互
- nanobot 可以通过聊天界面接受指令
- 支持自然语言命令（如"查看昨天的日志"、"推送代码到GitHub"）
- 能够自主决策并执行复杂任务链

## 技术栈
- **核心框架**：自定义AI助手平台
- **技能系统**：模块化技能（天气、记忆、定时任务等）
- **部署环境**：Linux服务器 + Python 3.12

# how to use
## linux deploy
进入虚拟环境 ```source venv/bin/activate```

将程序挂在后台运行  ``` nohup python -u app.py > output.log 2>&1 &```

web3信息流：```nohup python -u web3_app.py > web3.log 2>&1 &```

查看进程运行成功  ```ps aux | grep python```

关闭进程  ```kill [进程id]```

我进行测试查看效果 ```nohup python -u test.py > output2.log 2>&1 &```

获取有多少用户邮箱 ```wc -l emails.txt``` 结果 +1 就是最终的订阅用户数
